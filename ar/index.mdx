## أهلاً بك في Verify Group

**Verify Group Platform** هي واجهة API متعددة المستأجرين على مستوى المؤسسات لأجل:

- **KYC / التحقق من الهوية** — التحقق من الأفراد والشركات مدعوماً بـ الذكاء الاصطناعي ومصادر البيانات الموثوقة
- **إدارة المطالبات** — دورة حياة كاملة لمطالبات التأمين تشمل الاستقبال والتحقق والتحقيق ومسارات التسوية
- **فحص العقوبات ومكافحة غسل الأموال (AML)** — فحص قوائم المراقبة في الوقت الفعلي مع مراقبة مستمرة للقوائم العالمية
- **ذكاء الصوت** — تحليل بيومتري للصوت مدعوم بالذكاء الاصطناعي واكتشاف إشارات الاحتيال بالذكاء الاصطناعي
- **التحقق من صحة المستندات** — التعرف الضوئي على الحروف (OCR) والتحليل الجنائي للتحقق من المستندات

تتبع جميع الاستجابات الناجحة مغلفاً موحداً:

```json
{
  "success": true,
  "statusCode": 200,
  "message": "Success",
  "data": { ... },

    "requestId": "4515a20d-9157-8b5a-5b45-a4f111afc8ad",

}
```
