## تسجيل webhook

```bash
POST /api/v1/webhooks
```
```json
{
  "url": "https://your-system.io/webhooks/verify-group",
  "events": ["kyc.verified", "claim.status_changed", "sanctions.match.detected"],
  "secret": "whsec_your_signing_secret"
}
```

## التحقق من التوقيعات

كل تسليم يتضمن `X-VG-Signature-256`. تحقق دائمًا قبل المعالجة:

```typescript
import crypto from "crypto";

function verify(rawBody: string, signature: string, secret: string): boolean {
  const expected = `sha256=${crypto.createHmac("sha256", secret).update(rawBody).digest("hex")}`;
  return crypto.timingSafeEqual(Buffer.from(expected), Buffer.from(signature));
}
```

<Warning>حلّل الجسم كـ **بايتات خام** قبل التحقق — تحليل JSON أولاً يُغيّر تمثيل البايتات ويُفسد فحوصات التوقيع.</Warning>

## سياسة إعادة المحاولة

نُعيد محاولة عمليات التسليم الفاشلة بتراجع أسّي: 5 ثوانٍ ← 30 ثانية ← 5 دقائق ← 30 دقيقة ← ساعتان. بعد 5 إخفاقات، نضع علامة `failed` على webhook ونرسل لك تنبيهًا بالبريد الإلكتروني.

<Warning>يجب أن تستجيب نقطة النهاية خلال **30 ثانية**، وإلا نضع علامة فاشل على التسليم.</Warning>
