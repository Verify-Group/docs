---
title: المؤسسات
description: تعدد المستأجرين — كل مورد يقتصر نطاقه على مؤسسة.
---

كل مورد (المطالبات، المشاركون، طلبات KYC) ينتمي إلى **مؤسسة**. مرّر UUID الخاص بك في كل طلب:

```
X-Organization-ID: cfa70c25-1800-4748-97b0-ce675940a537
```

## تشغيل KYC لمؤسستك

```bash
POST /api/v1/organizations/{id}/run-kyc
```
```json
{"autoSync": true}
```

ينشئ تلقائياً مشاركاً من نوع أعمال، ويشغّل التحقق `business_ubo`، ويولّد سجلات مشاركين لكل مدير.
