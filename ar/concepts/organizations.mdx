---
title: المؤسسات
description: Verify Group متعدد المستأجرين. كل مطالبة ومشارك وطلب KYC مرتبط بمؤسسة تُعرَّف عبر ترويسة X-Organization-ID.
---

كل مورد (مطالبات، مشاركون، طلبات KYC) ينتمي إلى **مؤسسة**. مرّر UUID الخاص بك في كل طلب:

```
X-Organization-ID: cfa70c25-1800-4748-97b0-ce675940a537
```

## تشغيل KYC لمؤسستك

```bash
POST /api/v1/organizations/{id}/run-kyc
```
```json
{"autoSync": true}
```

يُنشئ تلقائيًا مشاركًا تجاريًا، ويُشغّل التحقق `business_ubo`، ويُولّد سجلات مشاركين لكل مدير.
