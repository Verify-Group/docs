---
title: المصادقة
description: قم بمصادقة طلبات Verify Group API باستخدام رموز JWT bearer، وتحديث رمز الوصول، وضبط ترويسة المؤسسة متعددة المستأجرين المطلوبة.
---

يجب أن يتضمن كل طلب `Authorization: Bearer <token>`.

## تسجيل الدخول

```bash
POST /api/v1/auth/login
```
```json
{"email":"user@example.com","password":"yourpassword"}
```

**الاستجابة:**
```json
{"data":{"accessToken":"eyJ...","refreshToken":"eyJ...","expiresIn":3600}}
```

## تحديث الرمز

```bash
POST /api/v1/auth/refresh
```
```json
{"refreshToken":"eyJ..."}
```

## الترويسات المطلوبة

| الترويسة | مطلوبة | الوصف |
|---|---|---|
| `Authorization` | دائمًا | `Bearer <token>` |
| `X-Organization-ID` | معظم نقاط النهاية | UUID الخاص بمؤسستك |

## أفضل ممارسات الأمان

<Check>احفظ رموز الوصول في الذاكرة، وليس في localStorage</Check>
<Check>احفظ رموز التحديث في ملفات تعريف الارتباط الآمنة httpOnly</Check>
<Check>استخدم HTTPS في كل مكان</Check>
<Check>قم بإبطال الرموز عند تسجيل الخروج</Check>
