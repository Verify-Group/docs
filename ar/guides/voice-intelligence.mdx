title: ذكاء الصوت
description: شغّل تحليل الصوت بالذكاء الاصطناعي على جلسات المكالمات، وفسّر درجات التوتر والخداع، وادمج الإشارات في درجة احتيال مركبة لكل مشارك.
---

## السيناريوهات

| السيناريو | حالة الاستخدام |
|---|---|
| `customerSupportComplaint` | مكالمات FNOL / الاستقبال (افتراضي) |
| `fraudSignalDetection` | التحقيق في الاحتيال / SIU |
| `debtCollection` | مكالمات التحصيل |

## تشغيل التحليل

```bash
POST /api/v1/voice/sessions/{sessionId}/analyze
```
```json
{"forceReanalysis": false}
```

## بنية النتيجة

```json
{
  "analysisResults": {
    "primaryScore": 62,
    "recommendedDecision": "MANUAL_REVIEW",
    "headline": "Moderate stress detected",
    "keyScores": {"stress": 68, "deception": 45, "coaching": 31},
    "importantCallSignals": [{"type": "elevated_stress", "confidence": 0.78}]
  }
}
```

## درجة الاحتيال المركبة

```bash
POST /api/v1/forensics/poi-fraud-check
```
```json
{"claimId": "claim-uuid"}
```

يُعيد درجات مركبة لكل مشارك تجمع بين إشارات الصوت وKYC والأدلة.

<Tip>عند إرجاع تحذير `SCENARIO_MISMATCH_NOT_SCORED`، تعيد الخلفية المحاولة تلقائيًا بالسيناريو المقترح.</Tip>
