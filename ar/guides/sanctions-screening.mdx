## مزامنة مصادر قوائم المراقبة

```bash
POST /api/v1/participants/sanctions-sources/sync
```

نقطة نهاية عامة — لا تتطلب مصادقة. استدعِها من مهمة مجدولة للحفاظ على تحديث المصادر.

## تشغيل فحص

```bash
POST /api/v1/participants/{participantId}/sanctions-check
```
```json
{
  "dataSources": ["UN_CONSOLIDATED", "OFAC_SDN", "EU_FINANCIAL_SANCTIONS"],
  "enableMonitoring": true
}
```

<Note>`dataSources` اختياري — احذفه للبحث في جميع المصادر المفعّلة.</Note>

## مستويات المخاطر

`clear` · `low` · `medium` · `high` · `critical`

## المراقبة المستمرة

عندما تكون `enableMonitoring: true`، فإنك تستلم webhooks عند ظهور تطابقات جديدة في أي قائمة مسجّلة.

| الحدث | المشغّل |
|---|---|
| `sanctions.match.detected` | تم العثور على تطابق جديد |
| `sanctions.match.resolved` | تم حل تطابق خاطئ |
