---
title: Organisations
description: Verify Group est multi-tenant. Chaque réclamation, participant et requête KYC est rattaché à une organisation identifiée par l'en-tête X-Organization-ID.
---

Chaque ressource (réclamations, participants, requêtes KYC) appartient à une **organisation**. Transmettez votre UUID dans chaque requête :

```
X-Organization-ID: cfa70c25-1800-4748-97b0-ce675940a537
```

## Exécuter un KYC pour votre organisation

```bash
POST /api/v1/organizations/{id}/run-kyc
```
```json
{"autoSync": true}
```

Crée automatiquement un participant entreprise, exécute une vérification `business_ubo` et génère des enregistrements de participant pour chaque dirigeant.
