---
title: Participants
description: Les participants représentent les personnes et entreprises liées aux réclamations et vérifications, y compris les demandeurs, témoins, bénéficiaires et parties assurées.
---

## Types

| Type | Description |
|---|---|
| `individual` | Personne physique |
| `business` | Entité juridique |

## Rôles sur une réclamation

`claimant` · `witness` · `third_party` · `beneficiary` · `agent` · `insured`

## Créer

```bash
POST /api/v1/participants
```
```json
{"firstName":"James","lastName":"Kamau","type":"individual","nationalId":"12345678","phone":"+254722000001","country":"KE"}
```

## Statut de vérification

`unverified` → `pending` → `verified` / `failed`
