## Bienvenue sur Verify Group

La **Verify Group Platform** est une API multi-tenant d'entreprise pour :

- **KYC / Vérification d'identité** — Vérification des particuliers et des entreprises propulsée par notre IA et des sources de données fiables
- **Gestion des sinistres** — Cycle de vie complet des sinistres d'assurance avec workflows de réception, validation, enquête et règlement
- **Screening sanctions & AML** — Screening en temps réel des listes de surveillance  avec surveillance continue
- **Voice Intelligence** — Analyse biométrique vocale et détection de fraude assistées par IA 
- **Authenticité des documents** — OCR et analyse forensique pour la vérification de documents

## Format de réponse

Toutes les réponses réussies suivent une enveloppe cohérente :

```json
{
  "success": true,
  "statusCode": 200,
  "message": "Success",
  "data": { ... },
  "metadata": {
    "requestId": "4515a20d-9157-8b5a-5b45-a4f111afc8ad",
    "timestamp": "2026-07-09T10:00:00.000Z"
  }
}
```

## Format d'erreur

```json
{
  "statusCode": 400,
  "message": "Validation failed",
  "error": "BadRequestException"
}
```
