---
title: Démarrage rapide
description: Obtenez vos identifiants API Verify Group, authentifiez-vous et effectuez votre premier appel KYC ou de réclamation en moins de cinq minutes avec des exemples curl prêts à coller.
---

## 1. Obtenez vos identifiants

Connectez-vous au [Dashboard](https://app.verify-group.io) et rendez-vous dans **Settings → API Keys**.

## 2. Authentifiez-vous

```bash
curl -X POST https://gateway.verify-group.io/api/v1/auth/login \
  -H "Content-Type: application/json" \
  -d '{"email":"your@email.com","password":"yourpassword"}'
```

**Réponse :**
```json
{
  "data": {
    "accessToken": "eyJhbGciOiJSUzI1NiJ9...",
    "refreshToken": "...",
    "expiresIn": 3600
  }
}
```

## 3. Incluez l'en-tête de votre organisation

```bash
export ACCESS_TOKEN="eyJhbGciOiJSUzI1NiJ9..."
export ORG_ID="cfa70c25-1800-4748-97b0-ce675940a537"

curl https://gateway.verify-group.io/api/v1/participants \
  -H "Authorization: Bearer $ACCESS_TOKEN" \
  -H "X-Organization-ID: $ORG_ID"
```

## 4. Exécutez une vérification KYC

```bash
curl -X POST https://gateway.verify-group.io/api/v1/kyc \
  -H "Authorization: Bearer $ACCESS_TOKEN" \
  -H "X-Organization-ID: $ORG_ID" \
  -H "Content-Type: application/json" \
  -d '{
    "participant_id": "<uuid>",
    "type": "individual",
    "scenario": "identity",
    "payload": {
      "identifier_type": "national_id",
      "identifier_number": "12345678",
      "country": "KE"
    }
  }'
```

<Check>Vous venez de réaliser votre première vérification d'identité !</Check>
