## Enregistrer un webhook

```bash
POST /api/v1/webhooks
```
```json
{
  "url": "https://your-system.io/webhooks/verify-group",
  "events": ["kyc.verified", "claim.status_changed", "sanctions.match.detected"],
  "secret": "whsec_your_signing_secret"
}
```

## Vérifier les signatures

Chaque livraison inclut `X-VG-Signature-256`. Vérifiez toujours avant de traiter :

```typescript
import crypto from "crypto";

function verify(rawBody: string, signature: string, secret: string): boolean {
  const expected = `sha256=${crypto.createHmac("sha256", secret).update(rawBody).digest("hex")}`;
  return crypto.timingSafeEqual(Buffer.from(expected), Buffer.from(signature));
}
```

<Warning>Analysez le corps sous forme d'**octets bruts** avant de vérifier — parser d'abord le JSON modifie la représentation en octets et casse la vérification de signature.</Warning>

## Politique de nouvelle tentative

Nous réessayons les livraisons échouées avec un backoff exponentiel : 5s → 30s → 5m → 30m → 2h. Après 5 échecs, nous marquons le webhook comme `failed` et vous envoyons une alerte par email.

<Warning>Votre point de terminaison doit répondre en **30 secondes**, sinon nous marquons la livraison comme échouée.</Warning>
