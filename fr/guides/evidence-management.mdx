title: Gestion des preuves
description: Téléversez les preuves de réclamation via le point de terminaison proxy, étiquetez les fichiers, exécutez les approbations par un relecteur et gérez les transitions de statut des preuves sur l'API Verify Group.
---

## Téléversement (recommandé pour les clients navigateur)

Route le téléversement via le backend pour éviter les problèmes de CORS :

```bash
POST /api/v1/evidence/public-upload/proxy-upload
Content-Type: multipart/form-data
```

Renvoie un `evidenceId` et une `signedUrl` pour un aperçu immédiat.

## Lister les preuves d'une réclamation

```bash
GET /api/v1/evidence?claimId={claimId}
```

## Étiqueter une preuve

```bash
PATCH /api/v1/evidence/{id}/tags
```
```json
{"tags": ["police_report", "verified", "high_priority"]}
```

## Approuver ou refuser une preuve

```bash
POST /api/v1/evidence/{id}/review
```
```json
{"status": "APPROVED", "notes": "Document authenticated against police database."}
```

| Statut | Signification |
|---|---|
| `PENDING` | En attente de relecture |
| `APPROVED` | Vérifiée et acceptée |
| `REJECTED` | Refusée |
| `UNDER_REVIEW` | En analyse forensique |
