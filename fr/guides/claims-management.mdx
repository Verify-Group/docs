---
title: Gestion des réclamations
description: Créez des réclamations d'assurance, faites évoluer leurs statuts, rattachez des participants et inspectez la piste d'audit tout au long du cycle de vie complet des réclamations Verify Group.
---

## Créer une réclamation

```bash
POST /api/v1/claims
```
```json
{
  "type": "MOTOR",
  "policyNumber": "POL-MT-KE-2025-8847",
  "description": "Rear-end collision on Mombasa Road",
  "incidentDate": "2026-06-15",
  "claimAmount": 185000
}
```

## Mettre à jour le statut

```bash
PATCH /api/v1/claims/{id}/status
```
```json
{"status": "INVESTIGATING", "reason": "Elevated fraud score"}
```

## Ajouter des participants

```bash
POST /api/v1/claims/{claimId}/pois
```
```json
{"personId": "participant-uuid", "role": "claimant"}
```

<Note>L'affectation d'un enquêteur fait automatiquement passer la réclamation à `INVESTIGATING`.</Note>

## Piste d'audit

```bash
GET /api/v1/audit/logs?resourceType=claim&resourceId={claimId}&sortBy=createdAt&sortOrder=desc
```
