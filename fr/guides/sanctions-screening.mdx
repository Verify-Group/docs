---

## Synchroniser les sources des listes de surveillance

```bash
POST /api/v1/participants/sanctions-sources/sync
```

Point de terminaison public — aucune authentification requise. Appelez-le depuis une tâche planifiée pour maintenir les sources à jour.

## Exécuter un contrôle

```bash
POST /api/v1/participants/{participantId}/sanctions-check
```
```json
{
  "dataSources": ["UN_CONSOLIDATED", "OFAC_SDN", "EU_FINANCIAL_SANCTIONS"],
  "enableMonitoring": true
}
```

<Note>`dataSources` est optionnel — omettez-le pour effectuer la recherche sur toutes les sources activées.</Note>

## Niveaux de risque

`clear` · `low` · `medium` · `high` · `critical`

## Surveillance continue

Lorsque `enableMonitoring: true`, vous recevez des webhooks lorsque de nouvelles correspondances apparaissent sur toute liste inscrite.

| Événement | Déclencheur |
|---|---|
| `sanctions.match.detected` | Nouvelle correspondance trouvée |
| `sanctions.match.resolved` | Faux positif résolu |
