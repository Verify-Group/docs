title: Vérification KYC
description: Exécutez des KYC individuels et d'entreprise via Verify Group, vérifiez le statut de vérification, synchronisez les résultats avec les participants et déclenchez un KYC à l'échelle de l'organisation.
---

## Exécuter un KYC

```bash
POST /api/v1/kyc
```
```json
{
  "participant_id": "uuid",
  "type": "individual",
  "scenario": "identity",
  "payload": {
    "identifier_type": "national_id",
    "identifier_number": "12345678",
    "country": "KE"
  }
}
```

## Vérifier le statut

```bash
GET /api/v1/kyc/{id}
```

| Statut | Signification |
|---|---|
| `pending` | En file d'attente |
| `processing` | En cours |
| `verified` | Identité confirmée |
| `failed` | Impossible de vérifier |
| `manual_review` | Nécessite une relecture humaine |

## Synchroniser le résultat vers le participant

```bash
POST /api/v1/kyc/{id}/sync
```

Met à jour le participant avec le nom vérifié, la date de naissance, l'adresse et la photo de visage extraite.

## KYC d'organisation

```bash
POST /api/v1/organizations/{id}/run-kyc
```
```json
{"autoSync": true}
```

Crée automatiquement des enregistrements de participant pour tous les dirigeants identifiés.
