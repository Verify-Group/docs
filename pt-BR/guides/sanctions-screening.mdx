---
title: Triagem de sanções
description: Faça triagem de participantes contra listas restritivas da ONU, OFAC e UE, ative monitoramento contínuo e receba alertas por webhook em novos matches de sanções.
---

## Sincronizar fontes de listas restritivas

```bash
POST /api/v1/participants/sanctions-sources/sync
```

Endpoint público — não requer autenticação. Chame a partir de um job agendado para manter as fontes atualizadas.

## Executar uma verificação

```bash
POST /api/v1/participants/{participantId}/sanctions-check
```
```json
{
  "dataSources": ["UN_CONSOLIDATED", "OFAC_SDN", "EU_FINANCIAL_SANCTIONS"],
  "enableMonitoring": true
}
```

<Note>`dataSources` é opcional — omita para pesquisar em todas as fontes habilitadas.</Note>

## Níveis de risco

`clear` · `low` · `medium` · `high` · `critical`

## Monitoramento contínuo

Quando `enableMonitoring: true`, você recebe webhooks sempre que novos matches aparecem em qualquer lista inscrita.

| Evento | Gatilho |
|---|---|
| `sanctions.match.detected` | Novo match encontrado |
| `sanctions.match.resolved` | Falso positivo resolvido |
