---
title: Gestão de sinistros
description: Crie sinistros de seguro, faça transições de status, vincule participantes e consulte a trilha de auditoria em todo o ciclo de vida na Verify Group.
---

## Criar um sinistro

```bash
POST /api/v1/claims
```
```json
{
  "type": "MOTOR",
  "policyNumber": "POL-MT-KE-2025-8847",
  "description": "Rear-end collision on Mombasa Road",
  "incidentDate": "2026-06-15",
  "claimAmount": 185000
}
```

## Atualizar status

```bash
PATCH /api/v1/claims/{id}/status
```
```json
{"status": "INVESTIGATING", "reason": "Elevated fraud score"}
```

## Adicionar participantes

```bash
POST /api/v1/claims/{claimId}/pois
```
```json
{"personId": "participant-uuid", "role": "claimant"}
```

<Note>Atribuir um investigador transiciona o sinistro automaticamente para `INVESTIGATING`.</Note>

## Trilha de auditoria

```bash
GET /api/v1/audit/logs?resourceType=claim&resourceId={claimId}&sortBy=createdAt&sortOrder=desc
```
