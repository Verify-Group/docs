---
title: Gestão de evidências
description: Envie evidências de sinistros pelo endpoint proxy, etiquete arquivos, execute aprovações de revisores e gerencie transições de status na API Verify Group.
---

## Upload (recomendado para clientes de navegador)

Roteia o upload através do backend para evitar problemas de CORS:

```bash
POST /api/v1/evidence/public-upload/proxy-upload
Content-Type: multipart/form-data
```

Retorna `evidenceId` e uma `signedUrl` para pré-visualização imediata.

## Listar evidências de um sinistro

```bash
GET /api/v1/evidence?claimId={claimId}
```

## Etiquetar evidências

```bash
PATCH /api/v1/evidence/{id}/tags
```
```json
{"tags": ["police_report", "verified", "high_priority"]}
```

## Aprovar ou recusar evidências

```bash
POST /api/v1/evidence/{id}/review
```
```json
{"status": "APPROVED", "notes": "Document authenticated against police database."}
```

| Status | Significado |
|---|---|
| `PENDING` | Aguardando revisão |
| `APPROVED` | Verificado e aceito |
| `REJECTED` | Recusado |
| `UNDER_REVIEW` | Em análise forense |
