---
title: Organizações
description: A Verify Group é multi-tenant. Cada sinistro, participante e requisição KYC está vinculado a uma organização identificada pelo cabeçalho X-Organization-ID.
---

Todo recurso (sinistros, participantes, requisições KYC) pertence a uma **organização**. Envie seu UUID em cada requisição:

```
X-Organization-ID: cfa70c25-1800-4748-97b0-ce675940a537
```

## Execute KYC para sua organização

```bash
POST /api/v1/organizations/{id}/run-kyc
```
```json
{"autoSync": true}
```

Cria automaticamente um participante empresarial, executa a verificação `business_ubo` e gera registros de participantes para cada diretor.
