## Linhas de negócio

`MOTOR` · `HEALTH` · `LIFE` · `PROPERTY` · `WORKERS_COMPENSATION`

## Ciclo de vida do status

```
DRAFT → SUBMITTED → UNDER_REVIEW → INVESTIGATING → PENDING_APPROVAL → APPROVED → SETTLED → CLOSED
                                                  ↘ PENDING_EVIDENCE / REJECTED
```

## Metadados de fluxo de trabalho

| Chave | Etapa |
|---|---|
| `metadata.intake[]` | Recepção FNOL |
| `metadata.validation[]` | Verificação de cobertura |
| `metadata.investigation[]` | Investigação em campo |
| `metadata.approval[]` | Decisão de aprovação |
| `metadata.settlement[]` | Confirmação de pagamento |
