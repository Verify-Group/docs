---
title: Tokens
description: Tempos de vida do token de acesso e do refresh token, estrutura do payload JWT e comportamento de rotação do refresh token na API da plataforma Verify Group.
---

| Tipo | Tempo de vida |
|---|---|
| Token de acesso | 1 hora |
| Refresh token | 30 dias |

## Payload decodificado

```json
{"sub":"user-uuid","email":"user@example.com","organizationId":"org-uuid","roles":["claims_manager"]}
```

Cada uso de refresh token **rotaciona** ambos os tokens — o par antigo é imediatamente invalidado.
