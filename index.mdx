## Welcome to Verify Group

The **Verify Group Platform** is an enterprise multi-tenant API for:

- **KYC / Identity Verification** — Individual and business verification powered by CreditInfo, InsightGenie, and proprietary AI
- **Claims Management** — Full insurance claims lifecycle with intake, validation, investigation, and settlement workflows
- **Sanctions & AML Screening** — Real-time watchlist screening via Sanctions.io with continuous monitoring
- **Voice Intelligence** — AI-powered voice biometric analysis and fraud detection using InsightGenie
- **Document Authenticity** — OCR and forensic analysis for document verification

## Base URLs

| Environment | URL |
|---|---|
| Production | `https://gateway.verify-group.io/api/v1` |
| UAT | `https://uat.gateway.verify-group.io/api/v1` |
| Local dev | `http://localhost:4008/api/v1` |

## Response format

All successful responses follow a consistent envelope:

```json
{
  "success": true,
  "statusCode": 200,
  "message": "Success",
  "data": { ... },
  "metadata": {
    "requestId": "4515a20d-9157-8b5a-5b45-a4f111afc8ad",
    "timestamp": "2026-07-09T10:00:00.000Z"
  }
}
```

## Error format

```json
{
  "statusCode": 400,
  "message": "Validation failed",
  "error": "BadRequestException"
}
```
