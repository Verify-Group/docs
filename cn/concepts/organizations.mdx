---
title: 组织
description: Verify Group 是多租户平台，每个理赔、参与人以及 KYC 请求都归属于由 X-Organization-ID 请求头标识的组织。
---

每一项资源（理赔、参与人、KYC 请求）都归属于某个**组织**。请在每次请求中传入你的 UUID：

```
X-Organization-ID: cfa70c25-1800-4748-97b0-ce675940a537
```

## 为你的组织发起 KYC

```bash
POST /api/v1/organizations/{id}/run-kyc
```
```json
{"autoSync": true}
```

会自动创建一个企业类型的参与人，运行 `business_ubo` 验证，并为每位董事生成参与人记录。
