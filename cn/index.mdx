## 欢迎使用 Verify Group

**Verify Group 平台**是一套企业级多租户 API，提供以下能力：

- **KYC / 身份验证** — 由 AI 和可信数据源 驱动的个人与企业身份验证
- **理赔管理** — 覆盖受理、审核、调查与结案的完整保险理赔生命周期工作流
- **制裁与反洗钱筛查** — 实时全球监控名单筛查与持续监测
- **语音智能** — AI 驱动的语音生物识别分析与欺诈信号检测
- **文档真伪核验** — 基于 OCR 与取证分析的文档验证能力

## 基础 URL

| 环境 | URL |
|---|---|
| 生产环境 | `https://gateway.verify-group.io/api/v1` |
| UAT | `https://uat.gateway.verify-group.io/api/v1` |
| 本地开发 | `http://localhost:4008/api/v1` |

## 响应格式

所有成功响应均遵循统一的封装结构：

```json
{
  "success": true,
  "statusCode": 200,
  "message": "Success",
  "data": { ... },
  "metadata": {
    "requestId": "4515a20d-9157-8b5a-5b45-a4f111afc8ad",
    "timestamp": "2026-07-09T10:00:00.000Z"
  }
}
```

## 错误格式

```json
{
  "statusCode": 400,
  "message": "Validation failed",
  "error": "BadRequestException"
}
```
