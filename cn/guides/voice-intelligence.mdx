## 场景

| 场景 | 使用场景 |
|---|---|
| `customerSupportComplaint` | FNOL / 受理通话（默认） |
| `fraudSignalDetection` | SIU / 反欺诈调查 |
| `debtCollection` | 催收通话 |

## 触发分析

```bash
POST /api/v1/voice/sessions/{sessionId}/analyze
```
```json
{"forceReanalysis": false}
```

## 综合欺诈评分

```bash
POST /api/v1/forensics/poi-fraud-check
```
```json
{"claimId": "claim-uuid"}
```

返回每位参与者的综合评分，融合了语音、KYC 与证据信号。

<Tip>当返回 `SCENARIO_MISMATCH_NOT_SCORED` 警告时，后端会使用建议的场景自动重试。</Tip>
