---
title: 语音智能
description: 对通话会话触发 AI 语音分析，解读压力和欺骗评分，并将各类信号汇总为参与人综合欺诈评分。
---

## 场景

| 场景 | 用途 |
|---|---|
| `customerSupportComplaint` | 报案 / FNOL 通话（默认） |
| `fraudSignalDetection` | SIU / 欺诈调查 |
| `debtCollection` | 催收通话 |

## 触发分析

```bash
POST /api/v1/voice/sessions/{sessionId}/analyze
```
```json
{"forceReanalysis": false}
```

## 结果结构

```json
{
  "analysisResults": {
    "primaryScore": 62,
    "recommendedDecision": "MANUAL_REVIEW",
    "headline": "Moderate stress detected",
    "keyScores": {"stress": 68, "deception": 45, "coaching": 31},
    "importantCallSignals": [{"type": "elevated_stress", "confidence": 0.78}]
  }
}
```

## 综合欺诈评分

```bash
POST /api/v1/forensics/poi-fraud-check
```
```json
{"claimId": "claim-uuid"}
```

返回按参与人的综合评分，综合语音、KYC 和证据等信号。

<Tip>当返回 `SCENARIO_MISMATCH_NOT_SCORED` 警告时，后端会自动使用建议的场景重试。</Tip>
