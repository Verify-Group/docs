title: 制裁名单筛查
description: 对照 UN、OFAC 和 EU 监控名单筛查参与人，开启持续监控，并在出现新的制裁命中时通过 webhook 接收告警。
---

## 同步监控名单源

```bash
POST /api/v1/participants/sanctions-sources/sync
```

公开接口 —— 无需认证。可通过定时任务调用以保持名单源新鲜。

## 发起筛查

```bash
POST /api/v1/participants/{participantId}/sanctions-check
```
```json
{
  "dataSources": ["UN_CONSOLIDATED", "OFAC_SDN", "EU_FINANCIAL_SANCTIONS"],
  "enableMonitoring": true
}
```

<Note>`dataSources` 可选 —— 省略则会搜索所有已启用的名单源。</Note>

## 风险等级

`clear` · `low` · `medium` · `high` · `critical`

## 持续监控

当设置 `enableMonitoring: true` 时，任一已订阅名单出现新命中都会触发 webhook 通知。

| 事件 | 触发条件 |
|---|---|
| `sanctions.match.detected` | 发现新的命中 |
| `sanctions.match.resolved` | 误报已处理关闭 |
