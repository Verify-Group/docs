---
title: KYC 验证
description: 通过 Verify Group 运行个人和企业 KYC，查询验证状态，将结果同步到参与人，并触发组织级 KYC。
---

## 发起 KYC

```bash
POST /api/v1/kyc
```
```json
{
  "participant_id": "uuid",
  "type": "individual",
  "scenario": "identity",
  "payload": {
    "identifier_type": "national_id",
    "identifier_number": "12345678",
    "country": "KE"
  }
}
```

## 查询状态

```bash
GET /api/v1/kyc/{id}
```

| 状态 | 含义 |
|---|---|
| `pending` | 已排队 |
| `processing` | 处理中 |
| `verified` | 身份已确认 |
| `failed` | 无法完成验证 |
| `manual_review` | 需人工审核 |

## 将结果同步到参与人

```bash
POST /api/v1/kyc/{id}/sync
```

用核验后的姓名、出生日期、住址以及提取的人脸照片更新参与人。

## 组织 KYC

```bash
POST /api/v1/organizations/{id}/run-kyc
```
```json
{"autoSync": true}
```

自动为所有识别到的董事创建参与人记录。
